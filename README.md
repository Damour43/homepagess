Mpuza is a job recruitment and networking dashboard interface designed to connect job seekers, recruiters, and companies.
This project provides a modern web-based dashboard where users can:

1. View profile statistics

2. Post jobs

3. Explore urgent hiring opportunities

4. Save favourite jobs

5. View job details with countdown timer

6. Manage company pages and groups
